#  Case Studies 

## Overview

This repository has three case studies that show how software development works in real life using Java, Python and SQL. Each project tries to solve a problem by designing a system handling data and making it work logically.

The Software Case Studies Portfolio is a collection of these projects.

## 1. Banking System Simulation (Java)

### Case Study Description

This project is like a bank where users can make accounts and do financial things. The system is made to work like a bank but in a simpler way using principles of Object-Oriented Programming.

The Banking System Simulation project uses Java.

### System Functionality

Here are the things the system can do:

* Users can. Log in with a username and password

* Users can make kinds of accounts:

* Savings Account

* Current Account

* Fixed Deposit Account

* Users can do things like:

* Put money in their account

* Take money out of their account

* Send money to accounts

* The system also has services like:

* Checking account balance

* Seeing transaction history

### Data Handling Approach

of using a big database the system uses files to store data:

* `users.txt` for login information

* `accounts.txt` for account details

* `transactions.txt` for transaction records

This way the data is saved even when the program is closed. The Software Case Studies Portfolio uses this method for the Banking System Simulation project.

### Design Approach

The system is made up of classes:

* Account classes that inherit from each other

* A Bank class to manage things

* A User class for logging in

* A Transaction class to track activities

The Banking System Simulation project uses these classes.

### Outcome

This project shows how Java can be used to make a bank system with clean and reusable parts. The Software Case Studies Portfolio includes this project.

## 2. CityCab Fare Calculator (Python)

### Case Study Description

This project is like a taxi fare calculator. It calculates how a ride costs based on distance what kind of car you want and what time you book it.

The CityCab Fare Calculator project uses Python.

### System Functionality

Here are the things the system can do:

* Users can input things like:

* How far they want to go

* What kind of car they want

* What time they want to be picked up

* The system calculates the fare based on:

* How far you go

* What time you book it

* The system also shows:

* A breakdown of the fare

* How much you have to pay in

### Additional Features

The system has a graphical interface made with Tkinter.

It can also remember your ride history.

You can see your rides in a table.

It shows you how rides you have taken and how much you have spent.

The CityCab Fare Calculator project has these features.

### Core Logic

The system adds money to the fare if you book during peak hours.

### Outcome

This project shows how Python can be used to make a program that interacts with users and does business logic. The Software Case Studies Portfolio includes this project.

## 3. Online Retail Sales Analysis (SQL)

### Case Study Description

This project is about analyzing sales data for a store using SQL. It makes a database and gets useful information out of it.

The Online Retail Sales Analysis project uses SQL.

### Database Structure

The database has four tables:

* Customers

* Products

* Orders

* Order_Items

These tables are connected to keep the data consistent.

### Data Analysis Performed

The system can do things like:

* Find the selling products

* Calculate how much each customer spends

* Look at sales for each month

* See how sales are distributed by category

* Find customers who have not ordered anything

* Calculate the order value

* Find the orders that make the money

### SQL Concepts Used

The system uses SQL things like:

* Joining tables

*. Averaging numbers

*. Ordering data

* Subqueries, for analysis

The Online Retail Sales Analysis project uses these SQL concepts.

### Outcome

This project shows how raw data can be turned into business information using SQL. The Software Case Studies Portfolio includes this project.
